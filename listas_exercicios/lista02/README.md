# Lista 02 - Exercícios em Java

Esta pasta contém a resolução dos exercícios da segunda listagem em Java de Exercicios, desenvolvidos para praticar aplicações de Programação Orientada a Objetos.

## Exercícios

### Exercício 01
Criação de uma classe com 5 atributos e 1 função, relacionada a um exemplo do cotidiano.  

Foi utilizado o tema **agendamento de exames**, ja que faz parte de meu cotidiano, com uma classe principal para instanciar o objeto, preencher os dados e exibir as informações formatadas na tela.

Para usar tem que se preencher cada uma das perguntas feitas, ao final da execução sera retornado os valores em ficha de agendamento.

**Arquivos:**
- `AgendamentoExame.java`
- `Main.java`

### Exercício 02
Programa que lê um número inteiro e imprime:
- seu antecessor
- seu sucessor

**Arquivo:**
- `Exercicio02.java`

### Exercício 03
Programa que lê os dados de 4 pessoas, contendo:
- nome
- peso
- altura

Ao final, calcula e exibe:
- o nome da pessoa com maior peso
- o nome da pessoa com maior altura
- a média dos pesos
- a média das alturas

Para usar preencher os dados de cada uma das pessoas, ao final da execução sera retornado os valores.

**Arquivo:**
- `Exercicio03.java`

### Exercício 04
Programa que lista os fatoriais de 1 a 10, exibindo no formato:

- O Fatorial de 1 é : 1
- O Fatorial de 2 é : 2
- O Fatorial de 3 é : 6

**Arquivo:**
- `Exercicio04.java`
